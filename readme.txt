== Description ==

Protect your pages by just adding the shortcode [login-logout-shortcode-simple]. No need to know programming. You can now choose which levels of users can access protected content. Install the new version and configure your preferences.

== Installation ==
* Install the plugin from the 'Plugins' section in your dashboard (Go to `Plugins -> Add New -> Search` and search for Login Logout Shortcode Simple).
* Alternatively, you can [download](https://downloads.wordpress.org/plugin/login-logout-shortcode-simple.zip "Download Login Logout Shortcode Simple") the plugin from the repository. Unzip it and upload it to the plugins folder of your WordPress installation (`wp-content/plugins/` directory of your WordPress installation).
* Activate it through the 'Plugins' section.
* Use the `Login Logout Shortcode Simple` menu to configure the plugin.
